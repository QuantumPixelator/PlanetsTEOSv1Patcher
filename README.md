# PLANETS:TEOS v1.0 Patcher

This repository contains tools for patching the PLANCFG.EXE file in Planets:TEOS v1.* BBS door game from the 1990s.

## Files
- `patch_cfg.c`: C source code for handling or generating patch configuration data.
- `patch_plancfg.py`: Python script for processing or managing patch plan configuration files.
## EXE
- A small patcher.exe file (compiled c source) is available in the Releases.
